C

C is a general-purpose, procedural programming language that was developed in the early 1970s. It is known for its efficiency and simplicity, and it is often used for systems programming, operating systems, and embedded systems. C has a small set of keywords and a structured syntax, which makes it easy to learn and use.

C++

C++ is an extension of the C programming language that was developed in the 1980s. It is a general-purpose, object-oriented programming language that is widely used for a variety of applications, including systems programming, operating systems, and graphical user interface (GUI) development.

SAD

System analysis and design refers to the process of studying a business or organizational system with the intention of designing a new or improved system that will enhance the organization's effectiveness and efficiency
